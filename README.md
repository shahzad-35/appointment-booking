## ‚öôÔ∏è Setup

```bash
git clone https://github.com/shahzad-35/appointment-booking.git
cd appointment-booking
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

Visit `http://localhost:8000`.
